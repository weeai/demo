# demo

React mobile h5

# getting started
```sh
# clone 项目
git clone https://github.com/weeai/demo.git

# 使用yarn作为node的项目管理
npm install -g yarn

cd demo

# 安装依赖
yarn

# 浏览器测试
yarn start

# visit http://localhost:3000

# 上线
yarn run build:all

# 上线的脚本和静态html
cd dist

```
